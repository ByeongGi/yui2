YUI 2 Archives

Documentation and Archives for YUI 2

YUI recommends using YUI 3.

YUI 2 has been deprecated as of 2011. This site acts as an archive for files and documentation.

Do not use YUI 2 for new projects. For information on the latest version of YUI please visit the YUI Library site.
