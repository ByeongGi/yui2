Re-establishing 1px Rounded corners in IE

This example provides the additional CSS you need to add to your page, after including Container Sam Skin CSS, in order to re-establish 1px rounded corners for IE. Support for 1px rounded corners was removed for IE in YUI 2.6.0 due to the functional side effects introduced by relative positioning and applying "layout", especially when a width was not provided for the Panel, as can be seen in "Panel2".
