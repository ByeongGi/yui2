Fonts CSS: Basic Test Suite for YUI Fonts

YUI Fonts CSS normalizes the font size and line height across all browsers. This example is a test page of HTML elements that allows you to explore the default rendering when YUI Fonts is applied.

View example in new window.

About This Example

The normalization provided by YUI Fonts CSS includes the following characteristics:

  • 13px Arial font
  • 16px line-height
  • A series of alternative fonts in case your chosen font is missing on the user's computer.
  • Percentages to use to set font sizes that are accurate and preserve flexibility.

Because CSS examples are susceptible to other CSS on the page, this example is only available in a new window at the above link.
